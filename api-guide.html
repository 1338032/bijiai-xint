§ 01 💡

基础概念，先搞清楚这几个词

🔑
API Key（密钥）
就像你的"通行证"，一串字母数字组合，是你调用 AI 服务的唯一凭证。绝对不能泄露给他人！
sk-xxxxxxxxxxxxxxxx
🌐
API 地址（Base URL）
告诉程序"去哪个服务器"请求 AI。官方地址固定，中转站会有不同的地址。
https://api.openai.com/v1
📦
模型名称（Model）
同一家服务商可能有多个 AI 模型，写清楚模型名才能调用到对的那一个。
gpt-4o / claude-3-5-sonnet
💰
Token（令牌）
AI 计费单位，大约每 1000 个汉字 = 1500 token。每次对话都会消耗 token，账户余额不足会报错。
约 750 字 ≈ 1000 token
🔁
中转站 / 二转站
国内访问不了官方 API？中转站架设在海外服务器，帮你转发请求，接口格式和官方一样。
relay / proxy
🎁
公益免费站
由热心网友维护，免费提供 API 调用。额度有限，适合轻度使用或测试体验。
free / public
安全提醒 API Key 是你的账号资产，请勿截图分享、勿存入公开文档、勿提交到 GitHub！一旦泄露请立即在官网撤销该 Key。
§ 02 🤖

OpenAI（ChatGPT）

🤖
OpenAI 官方 官方直连 需魔法上网
全球最强 GPT 系列，GPT-4o、o1、o3 等旗舰模型
OpenAI 是目前综合能力最强的 AI 平台，但国内无法直接访问，需要科学上网。新账号赠送少量免费额度，用完后需绑定境外信用卡付费。
  • 1开启科学上网（全局模式），访问 platform.openai.com
  • 2注册账号（建议用 Gmail，国内手机号无法注册）
  • 3进入 API Keys 页面 → 点击 Create new secret key
  • 4复制 Key（只显示一次！请立即保存）
  • 5填入 Base URL：https://api.openai.com/v1
# 常用模型名称
gpt-4o # 旗舰，速度与质量平衡（推荐）
gpt-4o-mini # 经济版，便宜好用
o3-mini # 深度推理模型
✓ 能力最强 ✗ 需科学上网 ✗ 需境外信用卡 $ 按量付费

Anthropic Claude

🧠
Anthropic 官方 官方直连 需魔法上网 写作首选
Claude Sonnet / Opus 系列，写作、创意能力出众
Claude 在长文写作、文风模仿、角色扮演方面表现突出，是创作类 APP 的热门选择。国内同样需要科学上网，且对 IP 地址要求较严格（部分地区的机场节点也会被拦截，建议使用美国/欧洲节点）。

  • 2注册并完成手机号验证（支持国内号码）
  • 3进入 API Keys → Create Key
  • 4复制 Key，保存妥当
  • 5Base URL 填：https://api.anthropic.com（注意：无 /v1）

✓ 写作创作强 ✗ 需科学上网 ✗ IP 要求严 $ 按量付费

中转站 / 二转站

什么是中转站？ 中转站架设在海外服务器，代替你转发 API 请求。你无需科学上网，只需把 Base URL 改成中转站地址，Key 用中转站提供的即可。接口格式和官方完全一致。
🔁
常见中转站类型 中转代理
收费型中转站，稳定可靠，支持 GPT / Claude / Gemini 多模型
中转站一般按 token 售卖，价格通常比官方略贵（包含代理服务成本），但免去了科学上网的麻烦。寻找时可以在各大 AI 交流群、论坛（如 V2EX、少数派、NodeSeek）搜索推荐。
  • 1在中转站注册账号，充值购买额度
  • 2在个人中心获取中转站提供的 API Key
  • 3将 Base URL 改为中转站的地址（如 https://api.xxx.com/v1）
  • 4模型名称通常与官方一致，直接填写即可
  • 5发送一条消息测试是否连通
选择中转站注意事项 ① 避免选择要求预付大额费用的站 ② 优先选口碑良好、运营超过 6 个月的 ③ 不要相信"永久有效"的承诺 ④ 先小额测试再充值 ⑤ 对话内容会经过第三方服务器，勿发送敏感信息
✓ 无需科学上网 ✓ 支持多种模型 ✗ 有跑路风险 $ 略贵于官方

公益免费站

公益站的由来 由热心网友自费搭建，免费开放使用。通常限速、限额度，适合轻度体验，不建议用于正式创作。请珍惜使用，不要滥用！
🎁
公益免费站特点 完全免费
无需付费，注册即用，额度有限
公益站大多基于 One API / New API 等开源项目搭建，接口与 OpenAI 格式完全兼容，支持 GPT-4、Claude、Gemini 等多种模型，每日有免费额度刷新。
  • 1在网络上搜索"OpenAI 公益 API"或加入相关 TG/QQ 群获取地址
  • 2注册账号（通常只需邮箱）
  • 3在"令牌"页面创建 API Key
  • 4填入对应的 Base URL 和 Key 即可
重要提醒 公益站随时可能停服，请勿依赖于重要创作。对话内容同样会经过服务器，切勿输入个人隐私、商业机密等敏感信息。
✓ 完全免费 ✓ 无需科学上网 ✗ 不稳定 ✗ 额度有限 ✗ 随时停服

§ 11 🔧

将 API 配置到「笔记AI」

获取到 API Key 和 Base URL 之后，按以下步骤填入笔记AI即可开始使用。

打开设置 → API 配置

点击底部导航栏的「设置」图标，找到「API 配置」或「AI 服务」入口。

填写 API 地址（Base URL）

将对应平台的 Base URL 粘贴到「API 地址」输入框。注意末尾是否需要 /v1，不同平台要求不同。

# 各平台 Base URL 速查
DeepSeek： https://api.deepseek.com/v1
通义千问： https://dashscope.aliyuncs.com/compatible-mode/v1
Gemini： https://generativelanguage.googleapis.com/v1beta/openai/
OpenAI： https://api.openai.com/v1
Claude： https://api.anthropic.com

填写 API Key

将你的密钥完整粘贴到「API Key」输入框，注意不要有多余的空格。

填写模型名称（Model）

在「模型」或「Model」输入框填入对应的模型名。也可以点击下拉列表选择（如果有的话）。建议先用较快的模型测试，确认连通后再换旗舰模型。

保存并测试

点击保存后，发送一条"你好"测试。如果 AI 正常回复，说明配置成功！如果报错，请检查 Key 是否复制完整、URL 格式是否正确。

§ 12 🛠

常见错误 & 解决方法

401 Unauthorized 认证失败

API Key 错误或无效。检查：① Key 是否复制完整（无空格）② Key 是否已被撤销 ③ Key 是否匹配当前 Base URL（不能用 DeepSeek 的 Key 去请求 OpenAI）

429 Too Many Requests 请求频率超限

请求太频繁或账户余额不足。解决：① 等待片刻再试 ② 检查账户余额 ③ 降低并发请求数量

404 Not Found 地址或模型不存在

Base URL 填写有误，或模型名称拼写错误。检查 URL 末尾是否多了/少了斜线、模型名是否准确。

连接超时 / 无响应 网络问题

国际官方 API 需要科学上网。解决：① 检查科学上网是否开启 ② 换全局模式而非 PAC 模式 ③ 尝试换节点 ④ 或改用国内平台/中转站

模型名称问题 不知道填什么

去各平台的官方文档查"模型列表"，或在平台控制台的模型广场里查找正确的 model ID。注意大小写和连字符要完全一致。
